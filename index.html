VERITAS Access
Решение в области программно определяемого хранения данных, оптимизированное для долговременного хранения информации.
В Access применяются интеллектуальные политики, позволяющие оптимизироватьразмещение данных в локальных средах или в частных / общедоступных облачных средах.Access повышает быстродействие, рентабельность и масштабируемость основного хранилища за счет использования многопротокольного доступа к данным, повышенияустойчивости и автоматизации перемещения данных. Это единственное решение LTR, тесно интегрированное с продуктом NetBackup — «золотым стандартом» в области защиты данных.

Для повышения динамичности бизнеса доступно два варианта развертывания Access: в виде устройства (развертывание «под ключ») и программно определяемого хранилища (модель BringYourOwnServer).
Access версии 7.4.2 предоставляет следующие основные возможности:
  • Интеграция Access с NetBackup для хранения резервных копий с дедупликацией

  • Обеспечение высокой доступности сервера резервного копирования

  • Глобальная дедупликация, охватывающая несколько серверов резервного копирования идоменов NetBackup

  • Решения для аварийного восстановления с использованием технологии NetBackup AIR

  • Поддержка дедупликации на клиенте (Client Direct) и на сервере резервного копирования

  • Поддержка шифрования NetBackup

  • Миграция данных из решений ODD и Cloud Catalyst в интегрированное решение на основе технологии дедупликации данных от Veritas

  • Новая модель лицензирования: по количеству ТБ с учетом емкости и срока действия подписки.

VERITAS NetBackup
Универсальная защита данных для предприятий, использующих облачные технологии
Защита данных на уровне хранилища. NetBackup интегрируем со средствами создания моментальных копий, репликации и NDMP от таких поставщиков, как Dell EMC®, HP, Hitachi, IBM®, NetApp® и др. Защита облаков. NetBackup поддерживает резервное копирование и восстановление элементов практически любой облачной среды путем размещения инфраструктуры NetBackup непосредственно в облаке.
  • Операционные системы: NetBackup интегрируется с десятками версий серверных операционных систем, включая Microsoft Windows®, Linux® и UNIX®.

  • Виртуальные системы: NetBackup интегрируется с ведущими гипервизорами, включая VMware vSphere® и Microsoft® Hyper-V.

  • Базы данных и приложения: NetBackup интегрируется с ведущими реляционными СУБД и платформами приложений, включая IBM DB2®, Microsoft® Exchange Server, Microsoft SQL Server®, MySQL™, Oracle®, SAP® и пр.

  • Большие данные: технология NetBackup Parallel Streaming позволяет выполнять резервное копирование масштабных систем, распределенных по множеству узлов, (таких как Hadoop) намного быстрее за счет параллельной передачи данных с каждого узла.
